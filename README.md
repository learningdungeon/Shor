# Shor
Research on Shor's algo
Quantum ECC Learning Journey 🔐⚛️
A comprehensive learning repository exploring the fascinating intersection of Elliptic Curve Cryptography and Quantum Computing. This project documents my journey through classical cryptography fundamentals and their quantum counterparts using Qiskit.

🌟 Overview
This repository contains my hands-on exploration of:

Classical Elliptic Curve Cryptography - Mathematical foundations and implementations

Quantum Computing Fundamentals - Quantum circuits, superposition, and entanglement

Quantum Threats to ECC - Understanding how quantum algorithms like Shor's threaten classical cryptography

Practical Implementations - From basic ECC operations to quantum circuit demonstrations

Basic understanding of cryptography and linear algebra
